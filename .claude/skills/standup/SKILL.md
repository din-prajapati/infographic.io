# Skill: standup

## Purpose

Generate a structured daily standup from the actual state of TASKS.md and STORY.md files.
No manual status updates required — Claude reads the files and derives the status.

---

## Protocol

### Step 1 — Scan All Story Files

Glob pattern: `docs/agile/epics/**/{STORY,TASKS}.md`

For each story directory found, read:
- `STORY.md` → Status field, ACs checked count, Closed date
- `TASKS.md` → Task checklist (checked vs unchecked)

### Step 2 — Classify Stories

| Status | Criteria |
|--------|----------|
| **Done (recently)** | STORY.md status = `✅ Done` AND Closed date ≥ 3 days ago |
| **In Progress** | STORY.md status = `🟡 In Progress` OR any TASKS.md task has `[x]` checked |
| **Ready to Start** | STORY.md status = `🔲 Not Started` AND Three Pillars Pre-flight all checked in TASKS.md |
| **Blocked** | TASKS.md has a task marked with `⏸` OR STORY.md has an open Issue in ISSUES.md |
| **Near DoD** | STORY.md has ≥ 80% of ACs checked |
| **Milestone Closing** | All stories in a milestone have status `✅ Done` or `🟡 In Progress` |

### Step 3 — Check Git Log (optional, if git is available)

```bash
git log --oneline --since="1 day ago" --pretty=format:"%h %s" 
```

Parse commit messages for `US-{DOMAIN}-{NNN}` references to confirm what actually shipped.

### Step 4 — Generate Standup Report

```markdown

|-------|-------|-----------|-------|
```

### Step 5 — Flag Anomalies

After the report, flag any of these if found:

```
⚠️ ANOMALIES

- {US-ID} has been "In Progress" for > 5 days with no new commits — may be stuck
- Milestone {M-ID} has 0 stories started but target date is within 7 days
- {US-ID} TASKS.md has no tasks filled in (Three Pillars incomplete) but status = In Progress
- {US-ID} STORY.md has no ACs written — not ready for implementation
```

---

## Quick Format (Short Standup)

If user asks for a quick version, produce just:

```
DONE:    {1-liner per closed story}
TODAY:   {1-liner per in-progress story}
BLOCKED: {1-liner per blocker, or "None"}
NEXT:    {top 2 stories ready to start}
```
